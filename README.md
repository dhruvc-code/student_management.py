# Student Management System

A console-based Student Management System built using Python and Object-Oriented Programming (OOP).

## Features

* Add Students
* View Students
* Search Student by ID
* Update Marks
* Delete Student
* Display Topper

## Concepts Used

* Classes and Objects
* Constructors
* Lists
* Loops
* Conditional Statements
* OOP Principles

## How to Run

```bash
python sms.py
```
